Example with shape-outside

With the css property shape-outside it possible to align text to shapes like the curve of this circle. There is currently no support for shape-inside, which requires in this case two shapes on left and right to align the text inside this circle. It's only working with float and fixed sizes, which is not very flexible and does not allow vertical centering :(
